# ComfyUI Flyway Plugin

一个专用的 ComfyUI 插件，包含图片目录操作和多行文本输入等功能。

## 功能特色

### 🐦‍🔥 Image List ↔ Directory
- 双向处理图片列表与目录（自动双模式）
- 根据输入自动判断是保存还是读取
- 支持路径设置、文件前缀、清空目录等功能
- 支持序列追加、跳过/限制数量

### 🐦‍🔥 多行文本轮询（顺序/随机）
- 支持三种输出模式：顺序、随机、索引
- 顺序模式：每次执行自动跳到下一行
- 随机模式：每次执行随机洗牌一行
- 状态缓存：跨执行保持进度
- 强制更新：确保每次执行都更新

## 安装方法

1. 在 ComfyUI Manager 中搜索 "comfyui-flyway"
2. 点击安装
3. 重启 ComfyUI

或者手动安装：
1. 将此仓库克隆到 ComfyUI 的 `custom_nodes` 目录
2. 重启 ComfyUI

## 使用说明

在 ComfyUI 中，您可以在 "flyway" 类别中找到这些节点。

## 注意事项

- 请确保设置的路径有足够的写入权限
- 清空目录功能会删除目录中的所有支持格式的图片文件
- 多行文本节点会自动忽略空行
- 顺序和随机模式会跨执行保持状态
